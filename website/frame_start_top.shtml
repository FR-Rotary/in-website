Welcome to the network resource server at Rotary and Vattentornet. The initial purpose of this server, "IN-maskinen", was to let people on this network gain access to their computers fom the outside, i.e. set up personal ftp/www servers. After the switch to public IP-numbers that is no longer necessary, the focus now is primarily to offer web-hosting, unix shell accounts and variuos server functions.
Everyone at Rotary and Vattentornet with an SGS internet connection can apply for an account

If you have any suggestions of server functions/services that you think should be implemented, send an email to IT responsible at FR. Please feel free to donate equipment to IN-maskinen. All contributors will be given credit on the hompage.
