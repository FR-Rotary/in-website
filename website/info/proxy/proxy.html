Proxy

IN-maskinen har nu en fullt fungerande SOCKS5 proxy-server. En proxy används för att länka ihop olika nätverk, t.ex. låta trafik passera en firewall med eller utan adressöversättning. Man kan även använda en proxy till att surfa anonymt, detta är dock ingen idé här eftersom all trafik loggas av netadmin. Program som kräver dubbelriktad kommunikation, t.ex. ICQ, AOL IM, IRC etc, har problem att fungera med adressöversättaren, genom att använda en proxy kan dessa fungera felfritt.

Proxyn kan användas fritt av alla på Rotary och Vattentornet, (inga portar blockerade).
Samtliga användare, interna och externa, kan utnyttja den som HTTP (WWW) och FTP proxy. Det går m.a.o att komma åt interna resurser utifrån utan att använda port-forwarding, observera att interna datornamn inte fungerar utifrån eftersom dessa inte finns i några externa DNS'er, (du måste använda lokala IP-nummer).

Proxy'n möjliggör för externa användare att komma åt interna FTP-servers även i passive mode. Eftersom FTP-data skickas över en icke förutbestämd port skulle detta kräva att proxyn var öppen på alla portar för externa användare. Eftersom detta skulle göra Rotary-nätet helt öppet för omvärlden är denna funktion avstängd.
Hör gärna av er med synpunkter om vad ni tycker om detta!

Viktig info angående Proxy'n

Vi har haft problem med att folk inte kan ansluta till proxy'n, detta beror bl.a. på att en del använder fel timout i t.ex. ICQ och liknande program och därmed avslutas inte tcp-anslutningarna korrekt. Observera att resultatet inte blir bättre av att sätta en lägre timout, använd värdet 15 min (900s) eller eventuellt 10 min (600s).
De användare som upprepade gånger låser upp proxyn genom dålig konfiguration kommer att bli underrättade per mail. I de fall som vederbörande inte har konto på IN-maskinen, (då vet vi inte vilken email personen har), kommer dennes IP-nr att blockeras på proxy'n. Om du råkar ut för detta så hör av dig per mail för att bli insläppt igen!

Följande kommando kan användas, (på in-maskinen), för att kolla status på proxy-uppkopplingarna
netstat | grep 1080

Om du har konfigurerat korrekt så skall det stå ESTABLISHED på din/dina förbindelser. Står det något som TIME_WAIT, FIN_WAIT2 eller liknande så använder du fel konfiguration.

Använd följande inställningar för proxyn

SOCKS5 proxy
Host: in.rotary.studenthem.gu.se, eller interna eller externa IP-numret
Port: 1080
IP: 193.11.234.4
Proxy timeout: 15 min

Användare på Rotary och Vattentornet kan använda proxy'n för all trafik, (inga portar blokerade). Övriga interna och externa användare kan bara använda proxy'n för HTTP och FTP, (port 80 och 21).

 

Inställningar för ICQ

Denna info är gammal och gäller inte sedan alla har riktiga IP-nr. ICQ fungerar nu utmärkt med normala inställningar utan proxy.

ICQ fungerar inte korrekt på Rotary-nätet, bl.a. fungerar inte chat, fileXfer, voicemail. Det går att få dessa funktioner att fungera om du initierar dem mot någon utanför Rotary-nätet ellor om den utomstående gör följande inställningar. Dock kvarstår problemet om båda sitter på Rotary-nätet.

Genom att använda en proxy-server kan alla dessa problem undanröjas. Observera att om ICQ skall funka inom Rotary-nätet måste båda parter använda proxy-servern.
